# Property Lead Manager

A full-stack SaaS platform built for real estate agents to capture, score, and manage property leads efficiently.

## Features

- **Lead Capture Forms** - Beautiful, mobile-responsive forms with unique tracking links per agent
- **Automatic Lead Scoring** - AI-powered scoring algorithm (0-100) based on budget, urgency, and financing status
- **Agent Dashboard** - Modern, professional dashboard with real-time analytics
- **Dark/Light Mode** - Toggle between themes for comfortable viewing
- **Charts & Analytics** - Visual performance tracking with Chart.js
- **Search & Filter** - Quickly find leads by name, phone, location, status, or score
- **Status Management** - Update lead status directly from the dashboard
- **Notes System** - Add timestamped notes to any lead
- **CSV Export** - Download all leads data for external analysis
- **Secure Authentication** - JWT-based auth with bcrypt password hashing
- **Multi-Agent Support** - Each agent gets unique tracking links and isolated data

## Tech Stack

**Backend:**
- Node.js + Express.js
- PostgreSQL (Supabase)
- JWT Authentication
- RESTful API

**Frontend:**
- Vanilla JavaScript (no frameworks needed!)
- Custom CSS with CSS variables for theming
- Chart.js for data visualization
- Responsive design (works on all devices)

**DevOps:**
- Git version control
- Environment-based configuration
- Ready for deployment on Render
